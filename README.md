# Ten Thousand Tokens Monitor Skill

Choose your language:

- [English](README.en.md)
- [中文](README.zh-CN.md)

## Quick Start

```bash
npm install
cp .env.example .env
npm run monitor
```

The monitor watches the Ten Thousand Tokens Ethereum mainnet `Launched` event and sends alerts only when a new token launch is detected.

## 快速开始

```bash
npm install
cp .env.example .env
npm run monitor
```

这个监控器监听 Ten Thousand Tokens 在 Ethereum 主网上的 `Launched` 事件，只有发现新代币发射时才会通知。
